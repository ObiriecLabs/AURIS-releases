# AURIS — Transcription Intelligence

**Download the latest version for macOS and Windows.**

> Source code is private. This repository contains official release binaries only.

---

## Download

Go to the [**Releases page**](https://github.com/ObiriecLabs/AURIS-releases/releases/latest) to download the latest version.

| Platform | File | Description |
|---|---|---|
| **macOS** | `AURIS-x.x.x-mac.dmg` | Drag AURIS.app to Applications |
| **Windows** | `AURIS-x.x.x-Setup.exe` | Recommended installer |
| **Windows** | `AURIS-x.x.x-win.zip` | Portable version |

## Requirements

**macOS:** macOS 11 Big Sur or later · Apple Silicon and Intel supported

**Windows:** Windows 10 (1803+) or Windows 11 · Microsoft WebView2 Runtime required
([Download here](https://developer.microsoft.com/microsoft-edge/webview2/) if missing)
